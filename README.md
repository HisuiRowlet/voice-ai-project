# 🎤 AI Voice Study Assistant

## 📌 Overview

This project is a simple AI-powered voice assistant that allows users to record speech, convert it into text, and receive an AI-generated response.

---

## ⚙️ Architecture

Frontend (HTML + JS)
→ Backend (Node.js + Express)
→ n8n (Workflow Automation)
→ Deepgram (Speech-to-Text)
→ NVIDIA Nemotron 3 Super (Free) (AI Response)
→ Response returned to frontend

---

## 🛠️ Tech Stack

* Frontend: HTML, JavaScript (MediaRecorder API)
* Backend: Node.js, Express
* Automation: n8n
* Speech-to-Text: Deepgram API
* AI Model: OpenRouter (NVIDIA Nemotron 3 Super (Free))

---

## 🚀 Features

* Record voice input from browser
* Convert speech to text using AI
* Generate intelligent responses
* End-to-end automated pipeline using n8n
